# 请识别音频的以下信息：
- genre：语音对话、直播语音、声优台词、音乐，如无法确信则留空
- character：如果为非语音对话，识别出具体主播或者声优名字，包括视频主播和虚拟主播以及游戏、动漫CV等，如无或者无法确信则留空
- source：如果是声优台词，请识别出作品名；如果是音乐，请识别出音乐名称，如无或者无法确信则留空
- text：识别语音的文字内容，如无或者无法识别则留空
- audio_description：从多角度描述音频内容，包括音频的氛围、情感、场景等，一段话纯文本，限制120个字

# 输出格式：
<caption genre="" character="" source="" text="">audio_description</caption>
